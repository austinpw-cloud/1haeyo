# 일해요 (1haeyo) — Surfaced 방법론 GTM 플랜

> Surfaced의 5단계 GTM 퍼널 적용:
> 1️⃣ 서비스 분석 → 2️⃣ 잠재 유저 정의 → 3️⃣ 길목 매핑 → 4️⃣ 채널별 전략 → 5️⃣ 실행 & 바이럴

---

## 1단계: 일해요 Product DNA

| 항목 | 내용 |
|---|---|
| **카테고리** | 온디맨드 노동 매칭 플랫폼 (gig work marketplace) |
| **핵심 기능** | 즉시 호출 + 예약 구인 매칭, 양방향 신뢰 시스템 |
| **해결하는 고통** | (구인) 알바 노쇼·인력난 / (워커) 연금 부족·고립감 |
| **타깃 유저** | 자영업 소상공인 + 시니어·부업자 (양면) |
| **분류** | B2B(소상공인) + B2C(시니어/부업자) 이중 구조 |
| **경쟁 포지션** | 일본 Timee + 당근 하이퍼로컬 + 시니어 친화 |
| **차별화** | 결정 마비 방지 설계, 실시간 매칭, 스킬 뱃지 |

### 일해요의 올바른 이해

```
"알바 구인구직 게시판"이 아님 (X)
→ "5분 안에 사람을 부르는 호출 시스템" (O)

"노인 일자리 복지 서비스"가 아님 (X)
→ "시니어의 경험을 자본으로 바꾸는 경제 플랫폼" (O)
```

## 2단계: 3층 유저 정의

Surfaced 방법론의 핵심: 유저를 **1차(핵심)·2차(확장)·대중(바이럴 매개체)** 3층으로 나눔.

### 구인자 측 (B2B)

| 유저 층 | 누구 | 핵심 고통 | 실제 언어 |
|---|---|---|---|
| **1차** | 분당 식당/카페 사장 (40~60대 자영업) | "알바 갑자기 안 나오면 하루 망함. 인력난 심각. 급하게 사람 구할 방법 없음" | "점심 피크인데 애가 안 나와요 ㅠㅠ 어쩌죠" |
| **2차** | 편의점/소매 매장 매니저, 프랜차이즈 점장 | "야간/주말 대타 구하기 너무 힘들다. 알바 앱에 올려도 반응 느림" | "주말만 하실 분 없나요" |
| **대중** | 자영업 관련 콘텐츠 소비자 | "자영업자들 고생한다는 얘기 공감하고 퍼뜨리고 싶다" | "이런 서비스 있었으면 우리 부모님 도움 됐을 텐데" |

### 워커 측 (B2C)

| 유저 층 | 누구 | 핵심 고통 | 실제 언어 |
|---|---|---|---|
| **1차** | 분당 거주 55~70세 시니어 (은퇴한 전문직/회사원) | "연금만으론 부족한데 매일 출근하긴 싫다. 심심하고 고립감" | "며칠 쉬고 하루이틀만 일할 데 없나" |
| **2차** | 부업 원하는 직장인, 백수/취준생, 경단녀 | "월급 부족. 본업 외 시간에 유연하게 돈 벌고 싶다" | "퇴근 후 주말만 알바 뛰고 싶어요" |
| **대중** | 시니어 자녀 세대 (30~50대) | "부모님이 집에만 계시는 게 마음 아프다. 의미있는 활동 했으면" | "엄마 아빠가 요즘 할 일이 없어서 우울해 하세요" |

### 이게 왜 중요한가

**"대중" 층이 바이럴 트리거**야. 자녀가 부모에게 "엄마, 이런 앱 있어"라고 추천하는 구조. 시니어는 스스로 앱을 찾지 않지만 **자녀의 한 마디**는 움직여.

## 3단계: 길목(채널) 매핑

각 유저 층이 **실제로 어디서 시간을 보내는지** 매핑.

### 구인자 길목

| 채널 | 누가 있나 | 어떤 대화가 오가나 |
|---|---|---|
| **네이버 카페 "아프니까 사장이다"** (회원 100만+) | 자영업 사장 전반 | "요즘 알바 구하기 진짜 힘들어요", "노쇼 때문에 미치겠다" |
| **네이버 카페 "자영업자 커뮤니티"** | 식당/카페 사장 | 인력 문제, 단기 채용 상담 |
| **배민사장님/쿠팡이츠 사장님 앱 내 커뮤니티** | 배달 받는 식당 | 인력·매출 고민 |
| **유튜브 "장사의 신", "사장학개론"** | 자영업 콘텐츠 소비자 | 운영·인력 노하우 |
| **인스타그램 해시태그** #자영업 #식당사장 #카페사장 | 자영업 운영 콘텐츠 | 일상·고민 공유 |
| **지역 상인회 단톡방** | 미금/야탑 실제 사장님 | 지역 이슈, 즉시 공유 |
| **블라인드** "자영업" 라운지 | 사장들 | 익명 고민 상담 |
| **네이버 지식인** | 초보 사장 | "알바 갑자기 안 나올 때 어떡하죠" 질문 |
| **프랜차이즈 본사 커뮤니티** (CU, GS25, 이디야 등) | 가맹점주 | 본사 공지 경로 |

### 시니어 워커 길목

| 채널 | 누가 있나 | 어떤 대화가 오가나 |
|---|---|---|
| **네이버 밴드 (분당 시니어 밴드 등)** | 지역 시니어 | 일상, 모임, 정보 공유 |
| **유튜브 시니어 채널** ("시니어 TV", "실버라이프") | 50~70대 | 재취업, 용돈 벌이 콘텐츠 |
| **유튜브 "은퇴 후 삶"** 관련 채널 | 은퇴자 | 자산 관리, 소일거리 |
| **카카오톡 오픈채팅** (분당 지역방, 50대+ 방) | 지역 시니어 | 정보 교환, 만남 |
| **당근마켓 동네생활** (분당구) | 분당 거주자 전반 | 동네 정보, 추천 |
| **네이버 카페 "분당맘"** | 30~60대 여성 (부모 포함) | 자녀+부모 동시 관심 |
| **경로당/복지관** (오프라인) | 70대+ | 집중된 시니어 커뮤니티 |
| **네이버 지식인** | 장년·시니어 | "은퇴 후 용돈 벌이" 검색 다수 |
| **조선·중앙·동아 실버 섹션** | 시니어 독자 | 재취업, 일자리 콘텐츠 |

### 청년/부업자 길목

| 채널 | 누가 있나 |
|---|---|
| **디시인사이드 알바 갤러리** | 알바/부업 관심 청년 |
| **블라인드 "N잡·부업" 라운지** | 직장인 부업러 |
| **에펨코리아, 네이트판** | 20~30대 일상 |
| **X/트위터 부업 계정** | 젊은 층 |
| **네이버 카페 "짠테크"** | 부업·절약러 |
| **유튜브 부업 채널** ("직장인 부업TV") | 세컨잡 관심 |

### 자녀 세대 (바이럴 매개체)

| 채널 | 누가 있나 |
|---|---|
| **맘카페** (분당맘, 미금맘) | 30~50대 여성 |
| **인스타그램 일상계** | 30~40대 |
| **X 부모님 얘기 계정** | 30대 자녀 세대 |
| **카카오스토리** | 40~50대 부모 세대 본인 |
| **블라인드** "가족" 라운지 | 직장인 자녀 |

## 4단계: 채널별 자연스러운 끼어들기 전략

**핵심 원칙: 광고 금지. 자연스럽게, 진정성 있게.**

### 구인자 채널 침투 전략

#### 4-1. 네이버 카페 "아프니까 사장이다"

**끼어들기 방법**: 제품 홍보 글 X. **고통 공감 + 해결 경험담** 형식으로.

```
제목: "알바 노쇼 때문에 하루 망친 썰... 다들 어떻게 대응하세요?"

내용: 오늘 점심 피크에 알바가 갑자기 안 나왔습니다. 전화도 안 받고.
제가 혼자 홀+주방 다 뛰었어요. 매출 반토막 났네요.
이런 경우 다른 사장님들은 어떻게 해결하세요?
주변 아는 분 있으면 급하게 부르시나요?
요즘 [일해요] 같은 거 있다던데 써보신 분 계세요?

→ 댓글에서 자연스럽게 실사용 후기 작성 (초기엔 자체적으로 유도)
→ 시간 지나면 진짜 사용자 후기가 쌓임
```

**규칙**: 홍보성 70% 이하 금지. 70%는 인사이트·고통·정보 공유.

#### 4-2. 유튜브 자영업 채널 협업

**접근 방법**: 큰 채널 X. **구독자 1~10만 중소 채널**을 공략. 응답률 높고 진정성 있음.

```
제안 이메일 템플릿:

"안녕하세요 OO 채널 구독자 XX입니다. 
최근 영상 중 '알바 노쇼 대응' 주제 영상을 보고 공감했습니다.
저는 분당에서 [일해요]라는 초단기 매칭 플랫폼을 운영하고 있는데,
이 문제를 해결하려 만들었습니다.
채널 구독자분들께 무료 체험 3개월 제공하는 콜라보 어떠신가요?
제품 협찬 + 실제 사용 사례 영상 제작 지원 드립니다."
```

#### 4-3. 네이버 지식인 GEO (AI 검색 최적화)

```
검색어: "알바 갑자기 안 나올 때"
→ 일해요 관련 답변 상위 노출
→ AI 검색(ChatGPT, Perplexity)도 네이버 지식인 답변 인용
→ 장기적 SEO + GEO 효과
```

**실행**: 자영업 관련 Q&A 50개에 자연스럽게 답변 (홍보 아닌 노하우 중심).

#### 4-4. 지역 상인회 단톡방 (가장 강력)

분당 미금/야탑 상인회를 이메일/전화로 컨택.
```
"이런 앱이 있는데 회원분들께 무료 체험 제공 가능합니다. 
상인회에도 월 소액 후원하고, 성공 사례 상인회 공지로 활용하셔도 됩니다."
```

### 시니어 워커 채널 침투 전략

#### 4-5. 네이버 밴드 — 가장 중요한 채널

시니어는 **네이버 밴드를 카톡만큼 써**. 분당에 "분당 시니어 모임" 같은 밴드가 수십 개 있음.

**접근법 2단계**:

```
1단계: 정보 공유 형식으로 글 작성
제목: "은퇴 후 용돈 버는 방법 정리해봤습니다"
내용: 기존 방법들 정리 + 마지막에 "요즘은 [일해요]라는 앱이 생겨서..."

2단계: 실사용 시니어 스토리 확보 후 재배포
"저희 아버지(67세) 이 앱으로 식당 홀서빙 하시는데 용돈도 벌고 건강해지셨어요"
```

#### 4-6. 유튜브 시니어 채널 협업

"시니어 TV", "실버라이프" 등 구독자 1~5만 시니어 채널과 협업.
```
콘텐츠 예시: 
"70세 영수 할아버지의 하루 - 일해요로 일하는 하루"
→ 실제 시니어 워커의 하루를 다큐 형식으로
→ 진정성 + 공감 + 자연스러운 제품 노출
```

#### 4-7. 당근마켓 동네생활 (분당구)

```
게시글 형식: "분당에 이런 앱 생겼네요"
- 제품 홍보 아닌 동네 정보 공유
- "시니어 분들 일자리 찾기 좋아 보여요"
- 댓글로 자연스러운 토론 유도
```

#### 4-8. 조선·중앙·동아 실버 섹션 PR

```
보도자료 앵글:
"70대 정년, 분당에서 시작된 시니어 경제의 실험"
→ 실제 워커 사례 기반
→ EBS 다큐프라임에서 다룬 일본 Timee 사례와 연결
→ 시니어 매체가 좋아할 스토리 구조
```

### 자녀 세대 채널 침투 (바이럴 매개체)

#### 4-9. 맘카페 "분당맘"

시니어 자체가 아니라 **자녀**에게 닿아야 부모에게 전파됨.

```
제목: "친정엄마한테 일해요 앱 설치해드렸는데 너무 좋아하세요"
내용: 엄마가 맨날 심심해하시고 TV만 보셨는데,
이 앱으로 근처 식당에서 하루 2~3시간 일하시니까 활기도 생기고 
용돈도 벌고... 우리한테 용돈 주실 정도예요 ㅋㅋ
추천드려요!

→ 자녀들이 공감 → 부모에게 전파
→ 분당맘은 회원 수만 명 단위, 도달력 강함
```

#### 4-10. 인스타그램 "부모님" 콘텐츠

```
릴스 콘텐츠 예시:
"아빠가 은퇴 후 만난 두 번째 인생"
→ 실제 워커 부자(父子) 인터뷰 릴스
→ 감동 + 공감 → 공유 → 자연 확산
```

## 5단계: 8주 내러티브 아크

Surfaced 방법론 핵심: **한 번에 때리지 말고 8주간 빌드업**.

### 전체 스토리 구조

```
Week 1-2: 씨앗 심기 (문제 공감)
Week 3-4: 문제 구체화 (일본 Timee 사례 소개)
Week 5-6: 제품 등장 (일해요 런칭)
Week 7-8: 소셜 프루프 (실제 사용자 스토리)
```

### 주차별 액션

#### Week 1-2: 씨앗 심기 (제품 언급 0%)

```
[구인자 타겟]
- 네이버 카페 "아프니까 사장이다"에 "알바 노쇼 썰" 공감 글
- 유튜브 쇼츠: "사장님이 겪는 3대 악몽" 콘텐츠
- 네이버 지식인: 자영업 고민 답변 달기

[워커 타겟]  
- 네이버 밴드: "은퇴 후 활동 추천" 정보글
- 유튜브 쇼츠: "70대인데 요즘 이렇게 삽니다" 감성 콘텐츠

→ 목표: 타겟 커뮤니티에 "이 사람/계정 괜찮네" 인식 심기
→ 제품 이름 언급 금지
```

#### Week 3-4: 문제 구체화 (제품 간접 언급)

```
[구인자 타겟]
- "일본에선 이런 앱으로 알바 문제 해결한대요" (Timee 소개)
- "한국에도 이런 거 나오면 좋겠다" 뉘앙스 유도
- 유튜브 분석 콘텐츠: "일본 Timee 성공의 이유"

[워커 타겟]
- "일본 70대 노인들이 앱으로 일한대요" EBS 다큐 재공유
- "한국엔 언제 이런 게 나올까요" 질문 던지기

→ 목표: "이런 서비스 필요해"라는 정서 형성
→ 실제로 분당 시장 조사 + 사전 관심자 리스트 수집
```

#### Week 5-6: 제품 등장 (Show, not Tell)

```
[공통]
- 브런치 에세이: "일해요를 시작했습니다" 창업 스토리
- Show HN / Product Hunt: 글로벌 커뮤니티 동시 런칭
- 언론 보도자료 배포

[구인자 타겟]
- 네이버 카페: "저번에 말씀드린 일본 Timee 같은 거 한국에도 생겼네요"
- 초기 사용자 인터뷰 영상

[워커 타겟]
- 네이버 밴드: "드디어 한국에도 이런 앱이"
- 분당 경로당/복지관 연계 설명 자료 배포

→ 목표: "기다렸던 그것"이라는 포지션
→ 이 시점에 실사용자 10~20명 확보 목표
```

#### Week 7-8: 소셜 프루프 (실사용자 중심)

```
[스토리 콘텐츠]
- "70세 영수 씨의 일해요 한 달 사용기"
- "분당 XX식당 사장님 - 노쇼 5번 해결한 이야기"

[데이터 콘텐츠]
- "런칭 2주, 이미 매칭 XX건"
- "시니어 워커 평균 시급 XX만원"

[바이럴 장치]
- "나의 일해요 경력 카드" 공유 기능 (시니어가 가족에게 자랑)
- "우리 가게 일해요 사용 배지" (구인자용 자랑)

→ 목표: 유기적 성장 시작
→ 이 시점에 실사용자 50~100명 + 월 매칭 30건+
```

## 6. 와우 모먼트 & 바이럴 트리거

Surfaced 방법론: **공유하고 싶은 순간**을 설계.

### 워커 와우 모먼트

```
"와, 앱 열고 10분 만에 일자리 잡혔다"
→ 시니어한테 이건 충격적 경험 (기존 대비)
→ 가족/친구에게 자랑하고 싶음
```

**공유 장치**:
- **"나의 일해요 이력 카드"**: 근무 횟수, 받은 별점, 스킬 뱃지를 담은 공유 카드
- 카톡/인스타로 공유 가능
- "우리 엄마 일해요 2개월 사용 후 받은 배지" 형식

### 구인자 와우 모먼트

```
"5분 만에 사람 왔다. 이거 진짜 되네"
→ 자영업자 커뮤니티에 자발적 후기 작성
```

**공유 장치**:
- **"일해요 공인 가게" 배지**: 가게에 붙일 수 있는 스티커/POP
- "우리 가게 일해요 사용합니다" → 손님들도 인식
- 자영업 카페에 자발적 후기

### 자녀 세대 바이럴 트리거

```
"우리 아빠/엄마가 일해요로 활기찾으심"
→ 인스타 릴스, 맘카페 후기
→ 눈물 찡한 스토리 → 자연 확산
```

**공유 장치**:
- **"부모님 일해요 사용기" 영상 템플릿** 제공
- 인스타 릴스 필터, 릴스 챌린지

## 7. AI 검색 최적화 (GEO)

Surfaced의 핵심 기법: ChatGPT, Perplexity에서 **인용되게 만들기**.

### 전략

```
타겟 질문 예시:
- "알바 갑자기 안 나왔을 때 어떻게 하나요?"
- "분당에서 단기 알바 구하는 법"  
- "은퇴 후 용돈 버는 방법"
- "70대도 할 수 있는 일"
- "일본 Timee 같은 한국 앱"
```

### 실행

1. **자체 블로그 운영**: 위 질문들에 대한 구조화된 답변 콘텐츠 30~50개 작성
2. **Wikipedia 한국어 "긱 이코노미" 문서**에 일해요 사례 추가
3. **나무위키 "초단기 알바" 문서**에 등재
4. **티스토리/네이버 블로그**에 SEO 최적화 포스팅
5. **FAQ 페이지**를 구조화 데이터(JSON-LD)로 마크업

### 측정

- ChatGPT에 "한국 초단기 알바 앱"이라고 물어봤을 때 일해요 언급되는가
- Perplexity 검색 결과에 포함되는가
- 월 단위로 추적

## 12. 즉시 시작할 것 3가지

이 문서 길지만, 당장 해야 할 건 3개:

### ✅ 1. 타겟 커뮤니티 10곳 가입 (오늘)

```
- 네이버 카페: "아프니까 사장이다", "분당맘", "자영업자 카페"
- 네이버 밴드: 분당 시니어 관련 3개
- 카카오 오픈채팅: 분당 지역 3개
- 당근 동네생활: 분당구
```

**단순히 가입만 하고 1주일 눈팅**. 커뮤니티 룰과 톤을 익히는 게 먼저.

### ✅ 2. "씨앗" 콘텐츠 3개 작성 (이번 주)

```
1. 네이버 카페 자영업 고통 공감 글 1개 (제품 언급 0)
2. 네이버 밴드 시니어 일상 공감 글 1개 (제품 언급 0)
3. 브런치 "왜 일해요를 시작하는가" 에세이 1개 (창업 스토리)
```

### ✅ 3. Surfaced 본 제품을 일해요에 적용 (가장 강력)

너 본인이 Surfaced 만드는 사람이잖아. **일해요 = Surfaced의 첫 번째 케이스 스터디**로 쓰는 게 가장 강력해.

```
"Surfaced로 일해요 런칭했더니 X주 만에 Y명"
→ Surfaced 영업 자료
→ 일해요 신뢰도 동시 상승
→ 일석이조
```
